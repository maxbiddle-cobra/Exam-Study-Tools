Exam Domains
The Cloud Digital Leader exam covers 6 domains.
Digital Transformation with Google Cloud 10%
Cloud concepts, digital transformation drivers, Google Cloud value proposition, cloud vs. on-premises.
Innovating with Data and Google Cloud 30%
Data value chain, BigQuery, data lakes, ML concepts, AI/ML Google products, Looker.
Infrastructure and Application Modernization 30%
Compute Engine, GKE, Cloud Run, App Engine, hybrid cloud, migration strategies, APIs.
Google Cloud Security 10%
Shared responsibility model, IAM, encryption, compliance, security products.
Scaling with Google Cloud Operations 10%
Cloud monitoring, logging, SRE, cost management, Cloud Operations suite.
Trust and Security with Google Cloud 10%
Google's approach to trust, compliance programs, data residency, digital sovereignty.
Exam Tips

• The actual exam has 60 questions in 90 minutes.

• Passing score is 70% or higher.

• Focus heavily on Data & AI (30%) and Infrastructure (30%).

• Understand use cases, not just product names.

• Know the difference between IaaS, PaaS, and SaaS.

• Google uses the shared responsibility model — know who manages what.
